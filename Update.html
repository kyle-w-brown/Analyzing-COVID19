Update for Final Project

Providing update for final project. So, far this has a linear and expontential regression using the STATWORX COVID19 API. This is a public API and can be accessed very simply following the commands.

API link: https://github.com/STATWORX/covid-19-api

Whats completed here: - API stream and link to source to access for R and Python - GBM grid AutoML model and found some rather interested results in the cross-validation summary below. - Simple linear regression - Exponential regression found

What needs done: - Expontential regression from Michigan model, maybe adjusted to reflect US. - Started an exponential regression of US at the very bottom to see if plausible. - Research the effectiveness of each plot from the models. If demonstrate colinearity, overfitting, underfitting, etc it can be added to the dashboard and part of our story and analysis. Even though these visualizations maybe statistically irrelavent, we can still use it to explain the progression of the the linear regression analysis.
